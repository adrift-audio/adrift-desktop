## Adrift desktop application

Adrift desktop application

### Deploy

```shell script
git clone https://github.com/adrift-audio/adrift-desktop
cd ./adrift-desktop
nvm use 16
npm i -g yarn
yarn
```

### Environment

The `.env` file is required, check [.env.example](.env.example) for details

### Launch

```shell script
yarn dev:dev
```
